#  Tienda de Celulares – Proyecto Web en PHP con XAMPP

Este es un proyecto web desarrollado como una tienda de celulares que permite visualizar productos desde la página principal y administrar su contenido mediante un panel de administración. El sistema incluye funcionalidades de CRUD de productos, CRUD de tipos de productos y un login para el administrador, todo implementado en PHP y ejecutado en XAMPP.

---

##  Funcionalidades principales

###  Página principal (Index)
- Muestra 4 productos destacados con:
  - Imagen
  - Nombre del celular
  - Precio
  - Descripción corta
- Diseño sencillo y funcional para visualización rápida.

---

##  Panel de Administración

###  Login de administrador
- Acceso protegido mediante formulario de login.
- Validación de credenciales desde la base de datos.
- Sesiones activas para restringir el acceso a las páginas administrativas.

---

##  CRUD de Productos
Permite gestionar completamente los productos visibles en la tienda:

- Crear nuevos productos  
- Leer la lista de productos  
- Actualizar información (nombre, precio, descripción y tipo)  
- Eliminar productos  

Incluye subida de imágenes y validaciones básicas.

---

## CRUD de Tipos de Productos
Módulo dedicado para gestionar categorías o tipos de celulares:

- Agregar nuevos tipos
- Editar tipos existentes
- Eliminar tipos
- Asociación de productos a un tipo

---

## Tecnologías utilizadas
- PHP 8+
- HTML5 / CSS3
- MySQL 
- JavaScript básico
- Bootstrap (opcional)
- XAMPP como servidor local y gestor de base de datos
